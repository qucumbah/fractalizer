Complex
This class models the behaviour of complex numbers
Class Complex encapsulates the data in complex-like objects: {x, y} - cartesian form {r, a} - polar form After creation only the cartesian form is stored
Creation
new Complex(cartesian)
Creates new Complex object using complex-like object in cartesian form
cartesian - complex-like object in cartesian form: {x, y}
Complex.fromPolar(polar)
Creates new Complex object using complex-like object in polar form
polar - complex-like object in polar form: {r, a}
